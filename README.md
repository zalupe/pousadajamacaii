# Pousada Jamacaii

Site estático da Pousada Jamacaii, pronto para hospedagem compartilhada na Hostinger.

## Publicação

Envie para `public_html`:

- `index.html`
- `chalet.css`
- `script.js`
- `robots.txt`
- `sitemap.xml`
- `.htaccess`
- `favicon.svg`
- a pasta `assets`

Antes de publicar:

1. Substitua `SEU-DOMINIO.com.br` em `robots.txt` e `sitemap.xml`.
2. Confirme o domínio nas metatags sociais, caso sejam adicionadas URLs absolutas.
3. Confirme a tag do Google Ads no `<head>` do `index.html`.

Os cliques em links de reserva enviam os eventos `airbnb_click` e `booking_click` para `dataLayer`. O link do Instagram envia `contact_click`.
